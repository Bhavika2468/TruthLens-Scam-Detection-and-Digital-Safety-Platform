# ScamShield AI

AI-powered Digital Trust and Cyber Safety Platform.

## What this repo contains
This repo will be scaffolded from scratch (monorepo-style) into:
- `apps/web` — Next.js (UI + API routes)
- `packages/db` — Prisma schema + client
- `packages/scanner` — Universal scan engine (rules-based v1; AI pluggable)
- `packages/types` — Shared DTOs/types

## Local development (after Phase 1 scaffold)
1. Install dependencies (root)
2. Configure environment variables
3. Run Postgres + Prisma migrations
4. Start dev server
